spring boot jar操作

栏目：Java        日期：2020-07-20    阅读：

1、spring boot 打包

在pom文件中添加打包插件

            <!-- 打包 -->
            <plugin>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-maven-plugin</artifactId>
                <executions>
                    <execution>
                        <goals>
                            <goal>repackage</goal>
                        </goals>
                    </execution>
                </executions>
                <configuration>
                    <mainClass>com.springboot.ApiApplication</mainClass>
                </configuration>
            </plugin>

然后执行 打包命令

mvn install
#或
mvn package

2、运行jar

java -Dspring.config.location=/xx/application.properties -jar xx.jar

3、解包

将jar包解压

jar -xvf xx.jar
unzip xx.jar -d ./

目录结构

- BOOT-INF
    - classes
    - lib
- META-INF
    - MANIFEST.MF
    - maven
- org
    - springframework
        - boot

MANIFEST.MF内容

Manifest-Version: 1.0
Implementation-Title: test
Implementation-Version: 1.0.0.Final
Built-By: travis
Implementation-Vendor-Id: com.springboot.test
Spring-Boot-Version: 2.1.0.RELEASE
Main-Class: org.springframework.boot.loader.JarLauncher
Start-Class: com.com.springboot.test.Application
Spring-Boot-Classes: BOOT-INF/classes/
Spring-Boot-Lib: BOOT-INF/lib/
Created-By: Apache Maven 3.5.2
Build-Jdk: 1.8.0_151
Implementation-URL: https://projects.spring.io

需要修改就直接编译替换类即可。 替换文件后，肯定要把文件再打包

4、打包

打包需要指定文件清单，也就是MANIFEST.MF内容，否则包不能用。

jar -cvfM xx.jar xx/MANIFEST.MF ./
jar -cvfM0 xx.jar ./
