Welcome to Jekyll!

栏目：        日期：2020-07-20    阅读：

1、安装jekyll

需要ruby

yum install ruby
gem install jekyll

2、创建jekyll项目

jekyll new blob

3、启动jekyll项目

cd blob/
jekyll server

访问 http://localhost:4000/

4、build 项目

jekyll build

将 _site 目录下文件放到服务器任意位置

scp -r ./_site/* root@xx.xx.xx.xx:/usr/local/nginx/html/blob/

5、配置nginx

        location / {
            root   /usr/local/nginx/html/blob;
            index  index.html index.htm;
        }

6、注意

访问时可能有丢失格式问题，主要是没有在_config.yml中配置正确的访问地址导致资源找不到。

baseurl: "/blob" # the subpath of your site, e.g. /blog
url: "http://xx.xx.xx.xx" # the base hostname & protocol for your site, e.g. http://example.com
